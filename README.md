# Employee Engagement Survey - CodeAlpha HRM Internship

This repository contains an Employee Engagement Survey project completed as part of the CodeAlpha HRM Internship.

##  Task Overview
The objective of this task was to design and analyze an employee engagement survey to understand workplace satisfaction and feedback.

##  Work Done
- Designed survey questionnaire   
- Collected sample responses  
- Analyzed employee feedback  
- Prepared summary of findings  

##  Outcome
This task helped in understanding employee engagement and how feedback is used to improve workplace environment.
